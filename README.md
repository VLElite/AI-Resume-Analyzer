# AI Resume Analyzer & Job Matcher

An AI-powered Resume Analyzer built using Python, Streamlit, NLP, and Groq AI to evaluate resumes against job descriptions, calculate ATS scores, identify skill gaps, generate interview questions, and provide AI-powered feedback.

## Features

* Resume Parsing (PDF/DOCX)
* ATS Score Calculation
* Resume Match Score
* Skill Gap Analysis
* Matching & Missing Skills Detection
* AI Resume Feedback using Groq LLM
* Interview Question Generation
* PDF Report Generation
* Interactive Dashboard UI with Custom Styling

## Tech Stack

* Python
* Streamlit
* Pandas
* Groq API
* ReportLab
* PyPDF2
* python-docx

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
